# video-digests

Structured video analyses generated by [mcp-content-pipeline](https://github.com/berkayyildirim/mcp-content-pipeline).

Each markdown file contains key takeaways, a TLDR, a social hook, and topic tags, extracted from YouTube video transcripts and analysed via Claude's API.

## Structure

```
content/videos/
├── 2026-03-10-oracle-openai-end-plans-to-expand-flagship-data-center-bloomberg-tech-3-9-2026.md
└── index.md
```

Files are named `{date}-{slugified-title}.md` and synced automatically via the `sync_to_github` MCP tool.

## Source

All content is generated from publicly available YouTube videos. The analysis pipeline is open source:

- PyPI: [mcp-content-pipeline](https://pypi.org/project/mcp-content-pipeline/)
- GitHub: [berkayyildirim/mcp-content-pipeline](https://github.com/berkayyildirim/mcp-content-pipeline)

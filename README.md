# daily-news

Structured video analyses and AI-generated infographics from Bloomberg Technology daily episodes, generated by [mcp-content-pipeline](https://github.com/berkayildi/mcp-content-pipeline).

Each markdown file contains key takeaways, a TLDR, a social hook, and topic tags, extracted from YouTube video transcripts and analysed via Claude's API. Accompanying `.png` files are comic-book style infographics generated via the Gemini API.

## Structure

```
content/videos/
├── 2026-03-23-stocks-bonds-fall-as-no-end-to-war-in-sight-bloomberg-businessweek-daily-3-20-20.md
├── 2026-03-23-stocks-bonds-fall-as-no-end-to-war-in-sight-bloomberg-businessweek-daily-3-20-20.png
└── index.md
```

Files are named `{date}-{slugified-title}.md` (with an optional matching `.png`) and synced automatically via the `sync_to_github` MCP tool.

## Source

All content is generated from publicly available YouTube videos. The analysis pipeline is open source:

- PyPI: [mcp-content-pipeline](https://pypi.org/project/mcp-content-pipeline/)
- GitHub: [berkayildi/mcp-content-pipeline](https://github.com/berkayildi/mcp-content-pipeline)

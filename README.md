# daily-news

Structured content digests from YouTube videos and X (Twitter) feeds, generated by [mcp-content-pipeline](https://github.com/berkayildi/mcp-content-pipeline). Each markdown file contains key takeaways, a TLDR, a social hook, and topic tags. YouTube analyses include comic-book style infographics generated via Gemini.

## Structure

```
content/
  youtube/
    2026-04-12-bloomberg-tech.md
    2026-04-12-bloomberg-tech.png
    index.md
  x-digest/
    2026-04-12-x-digest.md
    2026-04-12-x-digest.png
    index.md
```

YouTube files are named `{date}-{slugified-title}.md` with optional matching `.png`. X digests are named `{date}-x-digest.md`. All files are synced via the `sync_to_github` MCP tool.

## Source

All content is generated from publicly available YouTube videos. The analysis pipeline is open source:

- PyPI: [mcp-content-pipeline](https://pypi.org/project/mcp-content-pipeline/)
- GitHub: [berkayildi/mcp-content-pipeline](https://github.com/berkayildi/mcp-content-pipeline)
